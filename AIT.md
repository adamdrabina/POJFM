##### Types of computer graphics
1. Vector graphics
	- images are represented using geometrical objects such as lines, curves,...
	- you can manipulate with them how you want and the quality will be always same
2. Raster graphics
	- images are stored and displayed as a grid of pixels
	- if you manipulate with them, you will destroy their quality (for example if you zoom in)
3. 3D graphics
	- it is derived from 2D vector graphics and uses three-dimensional objects
	- it is used for computer simulations, computer-generated imagery (CGI), for industry, design,.....
##### Types of graphics software
1. Image manipulation or image editing
	- vector graphics: AutoCAD
	- raster graphics: Adobe Photoshop
2. Presentation software for slide shows
	- Microsoft Powerpoint
3. Animation for creating moving pictures uses 2D or 3D
	- HTML5
	- JavaScript
4. 3D computer graphics software
	 - includes 3D modeling and rendering software
	 - Blender
##### Color models
to create colors in computer graphics
1. RGB model
	- primary colors of red, green and blue
	- monitors, display screens
2. CMYK model
	- cyan, magenta, yellow and black
	- printers
##### Multimedia
- combination of text, graphics, animation, sound and video delivered by electronic media
- VLC media player
##### Computer publishing
1. Electronic publishing
	- production of electronic books
2. Desktop publishing (DTP)
	- enables to design and produce printout in the style of a book, magazine
	- you need software called page layout programs
	- the layout has columns, row, illustrations, photos, ....
