- ##### maska site
	- 255.255.224.0
- ##### broadcast adress
	- kdyz chceme oslovit vsechny zarizeni v siti
	- HOST PART MA SAME 1 (v binarne napsane IP adrese)
- ##### network adress
	- identifikator cele site
	- HOST PART ma same 0 (v binarne napsane IP adrese)
- kdyz na HOST PART dosadim 0, dostanu nejnizsi IP adresu => kdyz 1, dostanu nejvyssi IP adresu
- ##### default gateway
	- v kazde lokalni siti (kdyz ji chceme mit propojenou s jinou siti, internetem)
	- adresa routeru
	- pocitac posila packet primo na router, kdyz neco posila do jine site, internetu
