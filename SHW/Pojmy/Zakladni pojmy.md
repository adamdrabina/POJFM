- ###### Host
	- kazde zarizeni v siti, ktere ma IP adresu
- ###### Client
	- zarizeni, ktere vyuziva serverove sluzby
- ###### Server
	- poskytuje sluzby clientum
- ###### Peer to peer VS Client server
	1. Client server - prima komunikace mezi clientem a serverem
	2. Peer to peer - nelze poznat ktere zarizeni je server a client
