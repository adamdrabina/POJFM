###### AUTO_INCREMENT 
- u primarniho klice
- automaticke hodnoty
###### PRIMARY_KEY
- primarni klic
###### NULL / NOT NULL
 - nevlozena hodnota, nic
 - neni to nula
 - musime mit na primarnim klici
###### UNSIGNED
-  nezaporne
###### DEFAULT
- defaultni hodnota
- prednastavena vychozi hodnota je 0
###### ZEROFILL
- vyplneni nulami
###### UNIQUE
- nesmi se opakovat hodnoty
- neni to ale primarni klic
- napriklad: budu mit ID jako primarni klic a rodne cislo nastavim jako unique
